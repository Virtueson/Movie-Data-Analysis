# 🎬 Movie Data Analysis

This project explores and analyzes movie datasets to uncover insights about trends, ratings, popularity, and other key factors influencing movies.

---

## 📊 Overview

The goal of this project is to perform **exploratory data analysis (EDA)** on movie data and extract meaningful insights such as:

* Popular genres
* Revenue trends
* Rating distributions
* Relationships between features (e.g., budget vs revenue)

---

## 📁 Dataset

The dataset includes information such as:

* 🎥 Movie titles
* 📅 Release year
* ⭐ Ratings
* 💰 Budget & revenue
* 🎭 Genres
* ⏱ Runtime

---

## ⚙️ Project Workflow

### 1. Data Loading

* Imported dataset using **Pandas**
* Inspected structure and column types

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Converted data types where necessary

### 3. Exploratory Data Analysis (EDA)

* Summary statistics (`describe`)
* Distribution plots
* Correlation analysis
* Group-based analysis (e.g., by genre or year)

---

## 📈 Key Analysis Performed

* **Top-performing movies** by revenue
* **Most common genres**
* **Trend of movie releases over time**
* **Budget vs Revenue relationship**
* **Rating distribution analysis**
